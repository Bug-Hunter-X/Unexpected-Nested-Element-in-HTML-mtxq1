This is a common error, but it can be difficult to track down. It is caused by an unexpected nested element within the HTML structure that interferes with expected behaviour. This example illustrates a problem with improper nested elements resulting in unexpected rendering.

<div id="container">
  <p>This is some text.</p>
  <ul>
    <li>Item 1</li>
    <div> <!-- Unexpected nested div -->
      <li>Incorrectly nested list item</li>
    </div>
    <li>Item 3</li>
  </ul>
</div>
